What is a favicon?

A favicon is a small icon or collection of icons associated with a website, web page, or web application. It’s displayed within the browser tabs and bookmarks bar. The examples below show the favicons for Google, Reddit, and Squarespace within the browser tab.

Use this tutorial if you have an ICO formatted favicon. An ICO formatted favicon ends with the .ico file extension. If you have a PNG formatted favicon use this tutorial instead.

step 1:

Upload or Create Your Favicon

Make sure that you have an uploadable file format for any upload (JPEG or PNG) avaliable. Then use the “Click/drag” to upload your favicon

step 2:

Generate Favicon

Click the Upload button to generate the favicon zip file for your image (Upload is started automatically if image is dragged and dropped onto the upload area). This file is exported as a zip because we intent to keep quality should you decide to use them anywhere

step 3:

Download the Zipped file

To access the zip file, download the zip file to your local machine, Unzip the file. The folder contains the following files

  • favicon512x512.png
  • favicon192x192.png
  • favicon180x180.png
  • favicon32x32.png
  • favicon16x16.png
  • favicon.ico
  • site.webmanifest

Then place the folder in the root directory of your website

step 4:

Copy Favicon HTML code

Copy the HTML code using the copy button. This should be placed in the head of your HTML document.
