FAQs

How long does it take to create a favicon?

Just few minutes! The well designed templates,icons, and editing tools help you make a favicon with ease.

Do I have to register to use iconator?

You can directly go to the iconator editor to start your design. But when you make some changes you have to register and save your project to enable editing at any time

Can I upload images as favicons?

Yes, you can upload images as favicons. Just make sure you have the right file format

What file do I get in return?

You will get a zip file with all the files you need to use your favicon. You can upload this file to your website or download it and use it on your website.

Do I own the rights to my favicon?

Yes, you own the rights to your favicon. You can use it on any website you want.
